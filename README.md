# Weather App

This is a simple web application that allows users to check the weather information for a specific city. Users can enter the name of a city, and the app will display the current weather conditions including the temperature and a brief description. The application was made by Abdulmajeed.

## Usage

1. Open the Weather App by accessing the following URL: [Weather App](https://weather-app-mjeed91.000webhostapp.com/)

2. Enter the name of the city for which you want to check the weather in the provided input field.

3. Click the "Search" button to retrieve the weather information for the specified city.

4. The app will display the weather details including the temperature, weather condition, and a brief description.
